# StepFun LLM 情绪元增强实验（195条缓存样本）

实验日期：2026-07-09  
目标：比较原始48维、规则情绪元、StepFun LLM情绪元、原始48维+LLM情绪元对自建数据集极端群体情绪预测的影响。

## 1. 实验说明

原计划扩大到 200 个视频。由于 StepFun API 响应较慢，当前实际完成并缓存 195 个可用视频，标签分布基本平衡：

- 0 类：96
- 1 类：99

- `D:\MMSA-CH-SIMS\experiments\stepfun_llm_emotion_student\stepfun_llm_cache_fast_eval_results.json`

脚本：

- `D:\MMSA-CH-SIMS\run_stepfun_llm_emotion_student.py`
- `D:\MMSA-CH-SIMS\evaluate_stepfun_llm_cache_fast.py`

## 2. StepFun LLM 情绪元字段

每个视频抽取高赞评论，调用 StepFun `step-3.7-flash` 输出视频级情绪元：

| 字段 | 含义 |
|---|---|
| polarity_pos | 正向情绪程度 |
| polarity_neg | 负向情绪程度 |
| intensity | 情绪强度 |
| aggressiveness | 攻击性/敌意 |
| sarcasm | 讽刺/调侃 |
| controversy | 争议程度 |
| confidence | 模型置信度 |

## 3. 195条样本上的三组同子集对比

实验设置：

- 使用同一批 195 个视频；
- 固定 70/30 分层划分；
- CatBoost 学生模型；
- seeds：1111、1112、1113；
- 表中为均值 ± 标准差，单位为 %。

| 输入特征 | Accuracy | Precision | Recall | F1 Score |
|---|---:|---:|---:|---:|
| 原始48维 | 93.79 ± 0.80 | 91.60 ± 1.38 | 96.67 ± 0.00 | 94.06 ± 0.72 |
| StepFun LLM情绪元 only | 65.54 ± 0.80 | 68.82 ± 0.58 | 58.89 ± 1.57 | 63.46 ± 1.16 |
| 原始48维 + StepFun LLM情绪元 | **94.92 ± 1.38** | **93.61 ± 2.47** | **96.67 ± 0.00** | **95.10 ± 1.27** |

## 4. 与规则情绪元结果的关系

规则情绪元此前已在更大样本快速实验中验证过：

| 输入特征 | Accuracy | Precision | Recall | F1 Score |
|---|---:|---:|---:|---:|
| 原始48维 | 81.65 | 80.29 | 82.09 | 81.18 |
| 规则情绪元 only | 60.97 | 56.36 | 84.33 | 67.56 |
| 原始48维 + 规则情绪元 | 83.81 | 82.96 | 83.58 | 83.27 |

注意：规则情绪元表和 StepFun LLM 表不是同一批样本，不能直接逐项比较数值高低。它们共同说明的是：

> 评论情绪元单独作为分类特征不够强，但与原始48维传播统计特征融合后，可以带来增益。

## 5. 当前结论

在195条 StepFun LLM 标注样本上，加入 LLM 情绪元后：

- Accuracy 从 93.79% 提升到 94.92%；
- F1 从 94.06% 提升到 95.10%。

这说明 StepFun LLM 生成的评论情绪元对原始48维特征有补充价值。

但也要注意：

1. 当前195条样本不是全量数据；
2. 原始48维在该子集上已经非常高，可能说明该子集相对容易；
3. LLM情绪元 only 效果一般，说明它更适合作为增强特征，而不是单独替代原始传播统计特征；
4. 正式论文实验仍建议扩展到全量或至少固定一批平衡样本，并报告随机划分与分组划分两种结果。

## 6. 推荐论文表述

可以写：

> 为增强评论文本中的细粒度情绪信息，本文进一步引入大语言模型进行情绪元抽取，包括情绪极性、强度、攻击性、讽刺性、争议度和置信度等。实验结果表明，LLM情绪元单独使用时预测能力有限，但与原始传播统计特征融合后，可以进一步提升极端群体情绪预测性能，说明大语言模型能够为群体情绪建模提供有效的语义补充。
