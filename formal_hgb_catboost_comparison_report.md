# HGB 与 CatBoost 正式比较报告

日期：2026-07-10

## 1. 比较设置

- 样本：195 个带 LLM 情绪元的视频。
- 特征：48 维原始特征 + 7 维 LLM 情绪元 + causal Temporal 聚合特征。
- 两个模型使用完全相同的输入特征、标签和评估指标。
- 模型种子：1111、1112、1113。
- 指标：Accuracy、Precision、Recall、F1。
- 协议：5 折分层交叉验证、按时间前 70% 训练/后 30% 测试、发布者分组 70/30 测试。

## 2. 结果

| 评估协议 | HGB Accuracy | CatBoost Accuracy | HGB F1 | CatBoost F1 |
|---|---:|---:|---:|---:|
| 5 折分层交叉验证 | 93.33±3.60% | 92.48±4.89% | 93.56±3.42% | 92.71±4.61% |
| 时间 70/30 | 81.36±0.00% | 81.92±2.59% | 79.25±0.00% | 79.93±3.41% |
| 发布者分组 70/30 | 60.25±8.57% | 76.90±12.15% | 18.18±13.64% | 48.64±10.56% |

## 3. 解释

1. HGB 在随机分层交叉验证中平均高约 0.85 个百分点 Accuracy、0.85 个百分点 F1，但标准差较大，优势属于小幅优势。
2. 时间外推时 CatBoost 略高，说明两者对时间分布变化都比较敏感，HGB 没有稳定优势。
3. 发布者完全隔离时，CatBoost 明显优于 HGB。HGB 的发布者分组 F1 仅 18.18%，说明它可能更依赖训练集中的局部数值模式；CatBoost 的 F1 也有较大波动，因此该结果应视为泛化风险警告，而不是高性能结论。
4. 因此，原来固定随机划分上的 HGB 98.31% 不能作为论文最终主结果。它更适合写成“随机划分下的竞争模型结果”。

## 4. 当前决策

暂不把 HGB 替换为 CatBoost，也不预先宣称某个模型全局最优。论文中建议：

- 将 CatBoost 与 HGB 同时列为表格模型基线；
- 以 5 折交叉验证和发布者分组结果作为主要稳定性证据；
- 将随机 70/30 结果标注为探索性结果；
- 后续扩大到至少 500 个视频后重新比较，并报告按发布者分组的置信区间。
