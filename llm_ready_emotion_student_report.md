# 第一阶段试验：大模型辅助情绪元标注 + 轻量学生模型

实验日期：2026-07-09  
数据集：论文自建“极端群体情绪预测数据集”  
脚本：`D:\MMSA-CH-SIMS\run_llm_ready_emotion_student.py`  
结果：`D:\MMSA-CH-SIMS\experiments\llm_ready_emotion_student\results.json`

## 1. 实验目的

验证“大模型辅助情绪元标注 + 轻量学生模型”这条路线是否有价值。

当前本机尚未配置可直接调用的大模型 API 或本地情绪大模型，因此本次先做 LLM-ready 原型：

```text
评论文本
↓
情绪元弱标注函数 score_text()
↓
视频级情绪元聚合特征
↓
轻量学生模型 CatBoost
↓
极端群体情绪二分类
```

后续如果接入大模型，只需要替换 `score_text()`，训练流程可以保留。

## 2. 当前情绪元

本次先抽取以下视频级情绪元特征：

- 情绪极性；
- 情绪强度；
- 攻击性；
- 讽刺/调侃倾向；
- 置信度；
- 评论长度；
- 感叹/疑问表达；
- 按评论点赞数加权的情绪统计。

注意：当前版本是规则弱标签，不是真正 LLM 输出。

## 3. 快速实验结果

实验设置：

- 固定 80/20 分层随机划分；
- seed=42；
- CatBoost 500 iterations；
- 这是一轮可行性测试，不是最终论文三 seed 表。

| 输入特征 | Accuracy | Precision | Recall | F1 Score |
|---|---:|---:|---:|---:|
| 原始48维特征 | 81.65% | 80.29% | 82.09% | 81.18% |
| 仅评论情绪元 | 60.97% | 56.36% | 84.33% | 67.56% |
| 原始48维 + 评论情绪元 | **83.81%** | **82.96%** | **83.58%** | **83.27%** |

混淆矩阵：

原始48维：

```text
[[234, 54],
 [ 48, 220]]
```

原始48维 + 评论情绪元：

```text
[[242, 46],
 [ 44, 224]]
```

## 4. 初步结论

评论情绪元单独使用效果一般，但与原始48维统计特征融合后，Accuracy 从 81.65% 提升到 83.81%，F1 从 81.18% 提升到 83.27%。

这说明：

1. 评论文本中确实包含对极端群体情绪预测有用的信息；
2. “情绪元增强”方向有正向信号；
3. 如果把当前规则弱标签替换为 LLM 输出，理论上有继续提升空间；
4. 这条路线比直接在原48维特征上调 CatBoost 更有论文创新性。

## 5. 下一步建议

下一步不建议马上上复杂 GNN，而应先把情绪元标注升级成真正 LLM 版本：

1. 随机抽取每个视频的高赞评论、早期评论和争议评论；
2. 让 LLM 输出 JSON：
   - polarity；
   - intensity；
   - emotion category；
   - aggressiveness；
   - sarcasm；
   - cause；
   - confidence；
3. 聚合成视频级情绪元特征；
4. 跑同样的 CatBoost/MLP 学生模型；
5. 与当前规则弱标签版对比。

推荐论文表述：

> 初步实验表明，引入评论级情绪元信息可以增强原始传播统计特征的预测能力。当前规则弱标注版本已带来约 2 个百分点的 Accuracy/F1 提升，说明大模型辅助情绪元建模具有进一步研究价值。
